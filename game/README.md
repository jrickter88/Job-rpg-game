# Game-specific files

Everything under `game/` belongs to this particular game: JSON content, maps, scenes,
localization, audio, and art. Reusable rules and persistent-state models belong in
`src/Rpg.Core`; Godot adapters belong in `src/Rpg.Game`.

Directories are organized by authoring concern, not by a single giant scene tree.
Empty directories contain `.gitkeep` placeholders until their milestones begin.
